# 📊 Sales Performance Dashboard (Power BI)

## 📌 Overview
This project is an interactive Sales Dashboard built using Power BI to analyze business performance across different regions, categories, and time periods.

## 🎯 Key Features
- KPI metrics: Total Sales, Total Orders, Average Sales
- Sales trend analysis over time
- Regional and category-wise performance
- Top-performing products
- Interactive map visualization

## 🛠️ Tools Used
- Power BI
- Excel

## 📈 Insights
- Identified top-performing regions and categories
- Analyzed sales trends over multiple years
- Highlighted best-selling products
